## Future projects

- **tc-stockholm-equity** *(coming soon)*
  — Spatial analysis of public transport accessibility inequalities across Stockholm's 
  northern and southern districts using SL open data (GTFS via trafiklab.se). 
  The project will compute a stop-level accessibility score based on the number of 
  scheduled departures per hour within a 500m radius, and map the resulting disparities 
  across the city to identify underserved areas relative to population density.
